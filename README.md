# Jello Server

This application is a mock version of Trello API, where a client can use the provided RESTful APIs to generate lists.

## Requirements

  * `ruby 2.2 <~`
  * `rails 5.0 <~`

## Usage

### Lists

Queries the root

* `GET '/'`

Queries sdf

* `POST '/'`

### Cards

Queries the root

* `GET '/'`

Queries sdf

* `POST '/'`
